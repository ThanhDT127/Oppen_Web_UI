# 🚀 Open WebUI Docker Quick Start Guide

## Prerequisites

1. **Docker Desktop** - Installed and running
2. **API Keys** - Gemini API key (required), OpenAI API key (optional)

---

## ⚡ 5-Minute Setup

### Step 1: Configure Environment

```powershell
cd D:\Works\Open_Web_UI\Oppen_Web_UI

# Copy environment template
Copy-Item .env.example .env

# Edit with your API keys
notepad .env
```

**Required in `.env`:**
```ini
POSTGRES_PASSWORD=your_secure_password_min_16_chars
GEMINI_API_KEY=AIza-your-gemini-key
LITELLM_KEY=sk-litellm-master-key
OPENAI_API_KEY=sk-your-openai-key  # Optional
ADMIN_KEY=admin-key-change-me
SUBKEY_ADMIN=your-subkey-from-users-json
WEBUI_SECRET_KEY=webui-secret-min-32-chars
```

### Step 2: Start Docker Stack

```powershell
.\scripts\start_all.ps1
```

Wait 30-60 seconds for all services to start.

### Step 3: Access Open WebUI

Open **http://localhost:3000** in your browser.

## 🔧 Management Commands

```powershell
# Start all services
.\scripts\start_all.ps1

# Stop all services
.\scripts\stop_all.ps1

# Restart all services
.\scripts\restart_all.ps1

# View logs
.\scripts\logs.ps1 -Service all -Follow
.\scripts\logs.ps1 -Service open-webui -Tail 50

# Docker commands
docker-compose ps              # Service status
docker-compose logs -f         # All logs
docker-compose restart open-webui  # Restart specific service
```

## 🔍 Troubleshooting

### Container won't start
```powershell
docker-compose logs <service-name>
docker-compose down
docker-compose up -d
```

### PostgreSQL connection issues
```powershell
docker-compose exec postgres pg_isready -U openwebui_user -d openwebui
docker-compose restart postgres
```

### Reset everything
```powershell
docker-compose up -d
```
